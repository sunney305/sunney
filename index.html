關於我

我是來自台北的高中生，喜歡的運動是滑雪與打羽球，對資訊與科技有興趣，平時喜歡透過攝影記錄生活，也樂於與他人交流互動。 在高二時，我曾對未來方向感到迷惘，因此透過製作這個網站，重新整理自己的學習歷程、營隊經驗與興趣發展。 在回顧的過程中，我發現自己對資訊、實作與影像表達的熱情，也希望藉由這個網站，呈現我一路累積的成長與探索。

我的多元營隊

Minecraft X python 程式營

在國小時期報名參與 Minecraft X python 程式營，透過遊戲實際感受 python 程式， 使用 Python 控制 Minecraft 中的角色與環境，例如透過程式自動建造方塊、改變地形， 讓我初步建立對程式設計的基本認知，也對資訊領域產生了濃厚的興趣。

無人機科學營

也是在國小報名參加的營隊，課程內容主要是先透過樂高拼湊出無人機的外型， 並與老師學習主機板的程式設計，與如何接線控制無人機的飛行。
