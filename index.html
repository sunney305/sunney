關於我

我是來自台北的高中生，喜歡的運動是滑雪與打羽球，對資訊與科技有興趣，平時喜歡透過攝影記錄生活，也樂於與他人交流互動。 在高二時，我曾對未來方向感到迷惘，因此透過製作這個網站，重新整理自己的學習歷程、營隊經驗與興趣發展。 在回顧的過程中，我發現自己對資訊、實作與影像表達的熱情，也希望藉由這個網站，呈現我一路累積的成長與探索。

我的多元營隊

Minecraft X python 程式營

在國小時期報名參與 Minecraft X python 程式營，透過遊戲實際感受 python 程式， 使用 Python 控制 Minecraft 中的角色與環境，例如透過程式自動建造方塊、改變地形， 讓我初步建立對程式設計的基本認知，也對資訊領域產生了濃厚的興趣。雖然當時只停留在入門， 但這份好奇心延續至今，讓我在高中選擇進一步接觸網頁設計與 C++， 並持續探索資訊領域的更多可能性。

無人機科學營

也是在國小報名參加的營隊，課程內容主要是先透過樂高拼湊出無人機的外型， 並與老師學習主機板的程式設計，與如何接線控制無人機的飛行， 並在結束後自行體驗無人機的飛行。這個活動製作了屬於自己的無人機， 讓國小的我對程式設計有了興趣，並且激發了我對科技的熱情。

多元營隊參與紀錄

從小的我就對新事物不排斥，也很喜歡電腦與手作，因此常常利用寒暑假時間體驗各式各樣的營隊。 不同的活動帶給我不同的收穫，有的讓我對資訊產生興趣，有的讓我更喜歡動手實作， 有的則讓我在團體互動中學會交流與合作。這些經驗共同構成了我成長過程中重要的一部分。

攝影作品集

我很喜歡透過鏡頭記錄景色細節與畫面，也喜歡在拍攝過程中觀察光影變化與構圖 ，從中找到美感與創意的表達方式，雖然我的構圖與手法並不是專業的，但我很享受能拍到我認為好看的 照片。
